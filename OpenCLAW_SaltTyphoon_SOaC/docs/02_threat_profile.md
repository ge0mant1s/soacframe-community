# Threat Profile

Key observed behaviors (high level):
- Exploit public-facing edge devices and services
- Persist with SSH authorized keys and local account creation
- Tunnel traffic (e.g., GRE) to blend C2/exfil
- Clear logs and impair defenses to reduce visibility
- **NEW:** Target macOS devices for credential access and persistence

See: https://attack.mitre.org/groups/G1045/
