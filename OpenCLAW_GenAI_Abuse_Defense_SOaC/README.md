# OpenCLAW: GenAI & LLM Abuse Defense (GLAD)
## Defending Against AI-Accelerated Threats and Shadow AI

This package provides Security-as-Code (SOaC) artifacts to detect AI-driven reconnaissance, prompt injection, and unauthorized data exfiltration via LLMs.

### Contents:
- `/Detections`: KQL rules for AI API abuse and prompt injection patterns.
- `/Policies`: DLP and Shadow AI blocking templates.
- `/Playbooks`: Response workflows for AI-related breaches.
- `/Hunting`: Queries for AI-automated reconnaissance behavior.
