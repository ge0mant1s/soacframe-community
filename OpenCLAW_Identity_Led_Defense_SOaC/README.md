# OpenCLAW: Identity-Led Intrusion Defense (ILID)
## Based on 2025 Threat Landscape Trends

This package provides Security-as-Code (SOaC) artifacts to defend against identity-centric attacks, including Token Theft, MFA Bypass, and Lateral Movement from Cloud Identity to SaaS/IaaS.

### Contents:
- `/Detections`: KQL and Sigma rules for Entra ID and AWS.
- `/Policies`: Conditional Access and IAM policy templates.
- `/Playbooks`: Logic App/SOAR workflows for identity containment.
- `/Hunting`: Proactive queries for session anomalies.
